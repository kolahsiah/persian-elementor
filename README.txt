بسته فارسی ساز افزونه المنتور به همراه اضافه شدن فونت های فارسی محبوب در بخش تایپوگرافی تمامی ویجت های المنتور و همچنین بهبود ظاهر ویرایشگر با اضافه شدن فونت یکان برای زبان فارسی

== Description ==

بسته فارسی ساز افزونه المنتور به همراه اضافه شدن فونت های فارسی محبوب در بخش تایپوگرافی تمامی ویجت های المنتور و همچنین بهبود ظاهر ویرایشگر با اضافه شدن فونت یکان برای زبان فارسی

= ویژگی ها =

- **فارسی سازی افزونه المنتور**

تنها با نصب افزونه فارسی ساز، افزونه های المنتور، المنتور پرو  و چند افزونه زیر را فارسی کنید.

* Elementor
* Elementor Pro
* Dynamic Conditions
* Ele Skin
* Essential Addons Lite

- **قالب های آماده فارسی**

با نصب افزونه فارسی ساز می توانید قالب های آماده فارسی را از قسمت کتابخانه المنتور درج کنید.
این قالب ها شامل سربرگ، پاورقی، صفحه تکی نوشته ها، صفحه تکی محصول، قالب ویجت پست ها، و.... می باشد.

این قسمت هر روز بروزرسانی شده و قالب ها به مرور تکمیل خواهد شد.

- **تاریخ شمسی**

از ویجت فرم المنتور استفاده می کنید؟ در این ویجت یک فیلد تاریخ قرار دارد که تا قبل از این به صورت میلادی قابل استفاده بود. حالا می توانید با نصب افزونه فارسی ساز المنتور، تقویم میلادی را به شمسی تبدیل کنید.

- **فونت های فارسی**

یکی دیگر از ویژگی های جذاب این افزونه، اضافه شدن فونت فارسی به المنتور می باشد.
بعد از نصب افزونه فارسی ساز المنتور فونت های زیر به لیست افزونه اضافه خواهد شد.

= لیست فونت های رایگان =

- **IRANYekan**
- **Vazir**
- **Shabnam**
- **Gandom**
- **Parastoo**
- **Tanha**
- **Samim**
- **Nahid**
- **Sahel**
- **Estedad**
- **Mikhak**


= Pro =
برای مشاهده لیست 22 فونت محبوب فارسی به سایت زیر مراجعه کنید.
* نسخه پرو فارسی ساز [افزونه المنتور](https://elementorfa.ir/%d8%a7%d9%81%d8%b2%d9%88%d9%86%d9%87-%d9%81%d8%a7%d8%b1%d8%b3%db%8c-%d8%b3%d8%a7%d8%b2-%d8%a7%d9%84%d9%85%d9%86%d8%aa%d9%88%d8%b1/)

- **آیکون های ایرانی**

دیگر ویژگی افزونه فارسی ساز، اضافه کردن آیکون شبکه های اجتماعی و بانک های ایرانی به کتابخانه آیکون المنتور می باشد. 


- **بهبود ظاهر ویرایشگر المنتور**

با نصب این افزونه ظاهر ویرایشگر را با فونت ایران یکان زیباتر کنید. در حالت عادی از فونت پیش فرض سیستمی استفاده می شود که در زبان فارسی خوانایی خوبی ندارد.


= Support =

* پشتیبانی [افزونه المنتور فارسی](https://elementorfa.ir/)

1. Upload `persian-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
